# Redis CPP CLI Practice application

## Commands to run:

```bash
# Compile the server
g++ server.cpp -o ./server && g++ client.cpp -o ./client && ./server
```

### Run the client

```bash
#run client
./client
```

## Output

### In Client window:

```bash
❯ ./client
Server Says: World
```

### In Server window:

```bash
❯ ./server
Client Data: Hello
```
